Смена аккаунтов

Для записи времени смены аккаунтов, выполните следующие действия:

1. Запустить TelegramMaster (см. Запуск TelegramMaster).
2. Перейти в раздел "Настройки".
3. Запустить функцию "Смена_аккаунтов".
4. Ввести время.
5. Нажать "Готово".

После выполнения указанных действий программа автоматически запишет введенные данные в файл `TelegramMaster/user_settings/config.ini`.
